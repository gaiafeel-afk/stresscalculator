# Stress Calculator

Simple, technical-free stress calculator quiz with email-required result unlock.

## Features

- 8 plain-language stress questions
- Progress bar and friendly one-click answer cards
- Email is required before results are shown
- Instant score + stress level + short next-step guidance
- Mobile-friendly layout

## Run locally

Just open `index.html` in your browser.

## Live demo (GitHub Pages)

After pushing to `main`, the workflow in `.github/workflows/deploy-pages.yml` deploys the site to GitHub Pages.

Expected URL:

- `https://gaiafeel-afk.github.io/stresscalculator/`
